# Earthquake-Magnitude-Estimation

## Description
This project predicts earthquake magnitudes using Machine Learning techniques. It compares the performance of XGBoost and Random Forest on seismic event data to estimate earthquake magnitude accurately.
## Technologies Used
•	Python
•	Jupyter Notebook
•	Pandas
•	NumPy
•	Matplotlib
•	Seaborn
•	Scikit-learn
## Features
•	Data Cleaning and Preprocessing
•	Exploratory Data Analysis (EDA)
•	Feature Engineering
•	One-Hot Encoding
•	5-Fold Cross Validation
•	Model Evaluation
•	Feature Importance Analysis
•	Model Comparison
## Machine Learning Models
•	XGBoost 
•	Random Forest 
## Evaluation Metrics
•	MAE (Mean Absolute Error)
•	RMSE (Root Mean Squared Error)
•	R² Score
